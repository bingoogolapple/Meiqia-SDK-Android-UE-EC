:running:美洽 SDK 用户体验案例 - 电商:running:
============

**案例中演示了:**
* 自定义美洽聊天界面
* 获取未读消息
* 消息推送
* 指定分配客服 ID 上线
* 绑定用户 ID 上线
* 上传用户自定义信息
